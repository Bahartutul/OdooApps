Product Image On Sale Order Line + Optional Product Lines and Sale Quotation/Order Report Odoo Apps (Default image and Changeable)

This App supports for both community as well as enterprise version. This odoo app helps to print product image in sale report of odoo. With this module user can see images of product in the sale order line and optional product line, also user can change image from line. it is easier to identify product via image in Odoo ERP Module.

Features

Product Image on Sale Order Line and Changeable

You can see product image on sale order line and optional line and also can change it from line.

Product Image on Sale Report

You can print the product image with the sale report.

Settings

You can also hide image from quotation/order report by untick it.

Product Image on Sale Order Line

You can see product image on sale order line and also can change from line.

Product Image on Optional Product Line

You can see product image on optional product line and also can change from line.
